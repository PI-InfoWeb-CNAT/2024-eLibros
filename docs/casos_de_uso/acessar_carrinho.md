### 1. Resumo 
Esse caso de uso permite ao usuário acessar o seu carrinho

### 2. Atores

### 3. Pré-condições
Estar cadastrado e logado no sistema
  
### 4. Pós-condições
Após a execução deste caso de uso, espera que o sistema:
- Mostre ao usuário seu carrinho
